# Easynote a pocket notebook
**Swift 5**,
**Support device with IOS 13**

## Description

**ENG**
A pocket notebook that will help you save your notes. Implemented communication with Firebase to store your data.

**RUS**
Карманная записная книжка, которая поможет сохранить записи. Реализована связь с Firebase, для хранения ваших данных.
